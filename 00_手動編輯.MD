- 或手動編輯 `src/content/works/<slug>.md`

### 「我想加新的 tag 分類（讓客戶可篩選）」
1. 編輯 [src/content/tags.json](src/content/tags.json) — 加一筆：
   ```json
   { "slug": "your-slug", "name": "中文名", "group": "form|scene|occasion|kind", "description": "說明" }
   ```
   - `slug` = 內部識別碼（英文小寫 + 連字號）
   - `group` 決定篩選 UI 顯示在哪一排：
     - `form` 外型（捧花/花束/胸花/花籃/盆花/氣球/架構作品）
     - `scene` 場合（婚禮佈置/氣球佈置/花藝佈置/婚禮/告別式）
     - `occasion` 時節（求婚/畢業/紀念日/情人節/母親節/生日）
     - `kind` 類別（花藝商品）
2. 至少一件作品要 tag 它 → 篩選 chip 才會出現（避免無效篩選按鈕）
3. 替作品打 tag：admin 介面或編輯 `.md` 檔 `tags:` 陣列

### 「篩選 UI 怎麼運作」
- 點 chip = 開啟該 tag 篩選；再點一次 = 取消
- 多個 chip 同時啟用 = OR 邏輯（符合任一即顯示）
- 右上角「清除全部」按鈕 = 重置所有篩選
- 狀態列即時顯示「篩選中：X / 總數 件」

### 「手機 / 平板某處版面跑掉」
- 各檔案的 `@media (max-width: 960px)` 是平板斷點
- `@media (max-width: 640px)` 是手機斷點
- `@media (max-width: 380px)` 是超小機（僅 Header 有）

### 「想改首頁輪播」
- **改文字** → 開 [public/news/news.md](public/news/news.md) 編輯
- **換圖** → 把新圖檔丟進 [public/news/](public/news/)，命名為 `01.jpg`~`05.jpg`（或在 news.md 改 `image:` 欄位指定其他檔名）
- **改輪播速度** → [index.astro:27](src/pages/index.astro#L27) 把 `INTERVAL_MS = 5000` 改成想要的毫秒數（例如 3000 = 3 秒）
- **改動畫方向** → [index.astro] 找 4 個 `@keyframes sf-slide-*`，把 `translateX(-40px)` 與 `translateX(40px)` 正負對調即可反轉
- **改動畫時長** → 4 處 `0.5s var(--ease-stately)` 與 JS `ANIM_MS = 500` 一起改（總時長 = ANIM_MS × 2，包含滑出 + 滑進）
- **改回同步動畫**（不要序列式）→ JS `go()` 函式內把 Phase 2 的 setTimeout 拆掉，把 `current` 設 `active` 移到 Phase 1 同時
- **規格參考** → 完整說明在 [public/news/README.md](public/news/README.md)
